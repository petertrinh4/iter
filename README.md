# Iter

# Features

- Interactive route creation using an interactive map
- Walking route generation using OpenRouteService
- Real-time distance calculation
- Save and manage personal routes
- Delete saved routes
- Current location detection using the Geolocation API
- Dark mode support
- Secure user authentication with AWS Cognito
- Responsive user interface
